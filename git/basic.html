git-basic

Для того чтобы загрузить ваш код в ветку master репозитория git@example.com:example/test.git, вам потребуется выполнить несколько шагов. Вот подробная инструкция:

  1. Инициализация локального git-репозитория: Если у вас еще нет локального git-репозитория для вашего проекта, вы должны создать его. Для этого перейдите в папку с вашим кодом через командную строку и выполните следующую команду: 
              git init
  2. Добавление и коммит вашего кода: Добавьте все файлы и изменения в вашем проекте в индекс git и сделайте коммит. Используйте следующие команды: 
              git add .
          git commit -m "Ваш комментарий к коммиту"
  3. Добавление удаленного репозитория: Теперь вам нужно добавить удаленный репозиторий git@example.com:example/test.git в ваш локальный git-репозиторий. Выполните команду: 
              git remote add origin git@example.com:example/test.git
  4. Отправка кода на удаленный репозиторий: Загрузите ваш код на удаленный репозиторий ветку master: 
              git push -u origin master
    Эта команда отправит ваш код в ветку master удаленного репозитория. После этого вас попросят ввести учетные данные (логин и пароль) для доступа к удаленному репозиторию.
  5. Завершение: Теперь ваш код должен быть загружен в ветку master репозитория git@example.com:example/test.git. Вы можете проверить это, перейдя на страницу репозитория на GitLab или используя команду git log для просмотра истории коммитов.

Вот и все, ваш код теперь доступен в репозитории на ветке master.
